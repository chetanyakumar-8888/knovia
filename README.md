# Knovia — A New Star of Knowledge ⭐

AI-powered study platform for Class 6–12 CBSE students.

## 🌐 Live Demo
knovia-study.vercel.app

## ✨ Features
- 📚 NCERT AI Study Room (Class 6-12, all subjects)
- 🧠 AI Quiz Generator with timer
- 🔬 18 Interactive Virtual Labs
- 🤖 AI Doubt Solver & Revision Planner
- 👤 Auth + Profile + Stats (Supabase)

## 🛠️ Tech Stack
React • Tailwind CSS v4 • Groq AI • Supabase • Vercel

## 🚀 Run Locally
npm install
npm run dev

## 📝 Environment Variables
VITE_GROQ_API_KEY=
VITE_SUPABASE_URL=
VITE_SUPABASE_ANON_KEY=
